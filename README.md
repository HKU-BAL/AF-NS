# AF-NS
Identification of long-read novel sequences

# Prerequisites 
Make sure you have installed all of the following prerequisites on your machine：<br> 
•	minimap2 <br> 
•	NUCmer <br> 
•	porechop<br> 
•	NanoFilt<br> 
•	bedtools<br> 
•	kraken2<br> 
•	RepeatMasker<br> 

# Run
python AF-NS.py -kraken_db NAME -i input.fq -r ref.fa -o output_folder <br>
